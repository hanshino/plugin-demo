# Plugin Demo Project

## Usage

### Start web server

normal nodejs server
```bash
node index.js
```

specific port
```bash
APP_PORT=3000 node index.js
```

### Open ChatGPT

1. use GPT-4 model
2. open plugin dropdown list
3. select `plugin store`
4. click right bottom `Develop your own plugin`
5. enter `http://localhost:3000` to `Plugin URL`
